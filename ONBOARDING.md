### Skills to Know About
- `/usage` — Track your token usage and costs for the current session and billing period. Used heavily here — check it often to stay within limits.
- `/model` — Switch between Claude models (Opus, Sonnet, Haiku) mid-session. Useful when you need heavier reasoning for architecture vs. lighter for quick edits.
- `/effort` — Adjust how much thinking Claude applies. Lower effort = faster responses; higher = deeper reasoning for complex problems.
- `/plugin` — Manage MCP plugins. Use to enable/disable servers like claude-in-chrome.
- `/doctor` — Diagnose Claude Code setup issues (missing env vars, broken MCP connections, etc.).
- `/review` — Trigger a code review of the current branch's changes.

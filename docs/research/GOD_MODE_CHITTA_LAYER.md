## 🌙 Part 1: Pre-Sleep Seeding (Ratri Sankalpa)

The subconscious is most programmable in the **hypnagogic state** (between wake and sleep).

### Protocol:

```
1. Lie down, no phone, lights dim
2. 10 slow breaths
3. Mentally review today's key pattern:

   "Binary search... when target > mid, low = mid + 1...
    the invariant... the search space halves..."

4. Visualize yourself solving effortlessly
5. Set intention: "This is now part of me."
6. Fall asleep with the pattern in mind
```

### Why It Works:

- **Hippocampus → Cortex transfer** happens during sleep
- **Theta wave state** is highly suggestible
- Creates **samskaras** (deep mental impressions)

### Template:

```
Tonight's Seed:
Pattern: _______________
Key insight: _______________
Visual: See myself solving _______________
Intention: "I am a _______________"
```

## 🌅 Part 2: Morning Retrieval (Pratah Smaran)

First 20 minutes after waking — chitta is still close to surface.

### Protocol:

```
1. Stay in bed, eyes closed
2. Ask: "What did I learn yesterday?"
3. Retrieve WITHOUT looking at notes
4. If stuck, struggle (don't look up yet)
5. Only then verify
6. Any gaps = today's priority
```

### Why It Works:

- **Sleep consolidation test** — what stuck?
- **Active retrieval** strengthens neural pathways
- **Gaps reveal** incomplete integration

### Template:

```
Morning Retrieval (before phone):
□ Yesterday's pattern: _______________
□ Could I recall it? Y/N
□ Gaps noticed: _______________
□ Today's focus: _______________
```

## 🚫 Part 3: No-Reference Practice (Niralamba Abhyās)

The KEY to chitta programming — practice without ANY support.

### Protocol:

```
1. Close all tabs, notes, documentation
2. Set 25-minute timer
3. Solve from PURE MEMORY
4. If stuck: struggle, don't look
5. Only after timer: verify and fill gaps
```

### Progression:

| Week | Level        | What You Do        |
| ---- | ------------ | ------------------ |
| 1    | With hints   | Normal practice    |
| 2    | Notes closed | Solve, then verify |
| 3    | No reference | Pure recall        |
| 4    | Teach others | Ultimate test      |

### Why It Works:

- Forces **chitta activation** (not manas reading)
- **Struggle = myelin growth**
- Exposes **real** knowledge vs. recognition illusion

## 🌊 Part 4: Flow State Entry (Samadhi-like Focus)

When manas quiets and you're just "doing" — chitta is operating.

### Pre-Conditions:

```
- Low dopamine baseline (no scrolling before)
- Single task environment
- Clear goal
- Skill-challenge balance
- No interruptions possible
```

### Entry Protocol:

```
1. 2 min slow breathing
2. State intention: "Only this. Nothing else."
3. Start with easy warm-up (2-3 min)
4. Gradually increase difficulty
5. Notice when thinking stops, doing begins
6. That's flow — stay there
```

### Signs You're in Flow:

- Time distortion (hours feel like minutes)
- Effortless concentration
- No self-talk, just action
- Solutions arise without "thinking"

## 🔥 Part 5: Samskara Formation (Deep Impression Making)

Not all practice creates lasting impressions. **Samskara-forming practice** does.

### Requirements:

| Element        | What It Means      | How to Apply       |
| -------------- | ------------------ | ------------------ |
| **Intensity**  | Full attention     | No multitasking    |
| **Emotion**    | Feeling engaged    | Connect to meaning |
| **Repetition** | Multiple exposures | Spaced over days   |
| **Struggle**   | Edge of ability    | Slightly too hard  |
| **Resolution** | Completion feeling | Finish the problem |

### The Formula:

```
Samskara Strength = Attention × Emotion × Repetition × Struggle
```

### Anti-Patterns (Creates Weak Impressions):

- ❌ Passive watching without doing
- ❌ Copy-pasting solutions
- ❌ Solving too easy problems
- ❌ Distracted practice
- ❌ No emotional connection

The fastest path to chitta: **become** the skill.

### Daily Affirmations:


### Why It Works:

- **Ahamkara adopts the identity**
- Brain prioritizes identity-consistent behaviors
- Reduces resistance to practice

### The Switch:

```

TO:   "I am a DSA problem solver"

FROM: "I am studying hacking"
TO:   "I am a hacker"
```

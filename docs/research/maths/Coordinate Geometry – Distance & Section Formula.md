## 3. Novel Game Modality Blueprints

---

### 🎮 Game Concept 1: **“Grid Navigator”**

* **Target Material:**
  Distance formula, coordinate movement

* **Psychological Principle:**
  **Spatial Simulation + Embodied Cognition**

* **Core Interaction:**

  * Player moves from point P to Q on grid
  * Must:

    * Break movement into x-step + y-step

👉 Visual:

* Path splits into horizontal + vertical

* Then diagonal (distance appears)

* **Mastery Condition:**

  * Instantly visualizes distance as √(dx² + dy²)
  * No formula memorization needed

---

### 🎮 Game Concept 2: **“Ratio Divider”**

* **Target Material:**
  Section formula, internal division

* **Psychological Principle:**
  **Weighted Averaging + Proportional Reasoning**

* **Core Interaction:**

  * Line segment AB shown
  * Player places point P given ratio m₁ : m₂

👉 UI:

* Drag point along line

* Visual weights adjust position

* **Mastery Condition:**

  * Understands “closer to heavier weight”
  * Applies correct ratio direction

---

### 🎮 Game Concept 3: **“Shape Validator”**

* **Target Material:**
  Distance-based shape verification (triangle, square, collinear)

* **Psychological Principle:**
  **Pattern Recognition + Structural Validation**

* **Core Interaction:**

  * Given 3–4 points
  * Player must:

    * Identify shape using distances

👉 Example:

* Square → all sides equal + diagonals equal

* **Mastery Condition:**

  * Instantly recognizes geometric structure
  * No step-by-step calculation needed

---

### 🎮 Game Concept 4: **“Midpoint Balance Engine”**

* **Target Material:**
  Midpoint concept

* **Psychological Principle:**
  **Symmetry Detection + Averaging**

* **Core Interaction:**

  * Two points with weights
  * Player must:

    * Find balance point

👉 Visual:

* Like a seesaw

* Midpoint = equilibrium

* **Mastery Condition:**

  * Understand midpoint as average
  * Applies instantly

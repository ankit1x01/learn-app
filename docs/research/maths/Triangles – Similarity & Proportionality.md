## 3. Novel Game Modality Blueprints

---

### 🎮 Game Concept 1: **“Triangle Matcher”**

* **Target Material:**
  Similarity criteria (AAA, SSS, SAS), correspondence

* **Psychological Principle:**
  **Pattern Matching + Relational Mapping**

* **Core Interaction:**

  * Two triangles shown
  * Player must:

    * Match vertices (A ↔ D, etc.)
    * Select correct similarity rule

👉 UI:

* Drag lines connecting corresponding vertices

* **Mastery Condition:**

  * Instant correct mapping
  * No correspondence confusion

---

### 🎮 Game Concept 2: **“Parallel Line Engine”**

* **Target Material:**
  BPT & converse

* **Psychological Principle:**
  **Cause–Effect Reasoning + Structural Logic**

* **Core Interaction:**

  * Triangle with line inside
  * Player must decide:

    * Is line parallel?

OR

* Given ratios → draw correct parallel line

👉 Example:

* Adjust line until:
  AD/DB = AE/EC

* **Mastery Condition:**

  * Instantly links ratio ↔ parallel
  * No confusion in direction

---

### 🎮 Game Concept 3: **“Similarity Scanner”**

* **Target Material:**
  Detecting similar triangles in complex diagrams

* **Psychological Principle:**
  **Visual Search + Pattern Recognition**

* **Core Interaction:**

  * Complex geometry diagram
  * Player must:

    * Highlight all similar triangle pairs

👉 UI:

* Tap triangles → system validates

* **Mastery Condition:**

  * Detect hidden similarities quickly
  * No missed structures

---

### 🎮 Game Concept 4: **“Scale Transformer”**

* **Target Material:**
  Scale factor, proportional sides

* **Psychological Principle:**
  **Scaling + Ratio Encoding**

* **Core Interaction:**

  * Given triangle
  * Player scales it (2x, 3x, etc.)

👉 Visual:

* Triangle grows/shrinks

* Side ratios displayed

* **Mastery Condition:**

  * Understands scaling invariance
  * Predicts new side lengths instantly

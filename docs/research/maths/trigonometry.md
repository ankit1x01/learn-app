## 1. Cognitive Profile of the Chapter

This chapter (**Introduction to Trigonometry**) is a **foundational cognitive system**:

* **Relational Geometry (🔥 core)**
  → Angle ↔ side relationships in right triangle
* **Ratio-Based Thinking**
  → sin, cos, tan = relationships, not values
* **Invariant Pattern Recognition**
  → Ratios don’t change with triangle size
* **Symbolic + Spatial Dual Encoding**
  → Diagram + formula must sync

---

### 🧠 How the Brain Chunks This Chapter

The brain builds **3 core mental anchors**:

1. **Triangle Orientation System**

   * Opposite / Adjacent / Hypotenuse

2. **Ratio Engine**

   * sin = opp/hyp
   * cos = adj/hyp
   * tan = opp/adj

3. **Invariance Rule**

   * Same angle → same ratios (scale independent)

---

👉 Core Insight:

> Trigonometry is NOT formulas
> It is **angle → shape → ratio mapping**

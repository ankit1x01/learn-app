Neuroscience & Psychology of Learning — Deep Cut

--- Memory Consolidation (How Memories Actually Form)
Synaptic tagging and capture (STC) - Learning creates a "tag" at a synapse; within ~2h a second learning event can "capture" it and strengthen both

- Implication: study two related topics within 2h of each other — they consolidate together

Memory reconsolidation

- Every time you retrieve a memory, it becomes labile (unstable) for ~6h before re-storing
- This is a feature: retrieval + correction rewrites the trace more accurately than re-reading
- What CHITTA's MCQ sessions exploit — wrong answer + immediate feedback edits the memory

Hippocampal replay

- During NREM sleep, hippocampus replays the day's learning at 10-20x speed to neocortex
- Sharp-wave ripples (~80-100ms bursts) transfer episodic memory to long-term semantic memory
- Practical: last input before sleep matters most — it gets replayed the most

Attention & Encoding Depth

Levels of processing (Craik & Lockhart, extended)

- Shallow: visual/phonological ("does it rhyme?") → weak trace
- Deep: semantic/self-referential ("how does this relate to what I know?") → durable trace
- Deepest: survival processing — framing information in terms of threat/survival activates amygdala → stronger hippocampal binding

Prediction error and dopamine

- When an outcome violates expectation, dopaminergic neurons fire → norepinephrine release → stronger LTP (long-term potentiation)
- Getting an answer wrong before seeing the right one produces a bigger dopamine signal than getting it right
- This is why pre-testing (attempting questions before learning) outperforms reading first

Attention gating via locus coeruleus (LC-NE system)

- Norepinephrine from LC acts as a "signal booster" — high arousal = stronger encoding
- Optimal arousal follows inverted-U (Yerkes-Dodson): moderate stress = peak encoding, too high = encoding shuts down
- Cortisol timing: morning cortisol spike (6-9am) primes encoding; afternoon dip is worst time for new material

Forgetting is Not Failure

Bjork's "new theory of disuse"

- Memory has two dimensions: storage strength (permanent, never decreases) and retrieval strength (context-dependent, decays)
- You never truly forget; you lose retrieval access
- Forgetting slightly before review maximizes reconsolidation depth — this is why FSRS targets R=85%, not 100%

Desirable difficulties (Robert Bjork, UCLA)

- Conditions that slow apparent learning but improve long-term retention:
  - Spacing (harder to recall = better consolidation)
  - Interleaving (harder to identify patterns = stronger discrimination learning)
  - Variability of practice (harder to perform = more robust schema)
  - Generation (harder to produce = deeper processing)

Retrieval Practice — The Most Replicated Finding in Cognitive Psychology

Test-enhanced learning (Roediger & Karpicke, 2006 + replications)

- One retrieval practice > three re-study sessions on delayed tests (1 week+)
- Free recall > cued recall > recognition — harder retrieval = better trace

The forward testing effect

- Testing on material A improves learning of subsequent material B (not just A)
- Mechanism: retrieval clears working memory of interference, improves encoding of new material

Retrieval-induced forgetting (RIF)

- Practicing some items suppresses related competing items (inhibition-based forgetting)
- Strategic implication: test high-priority concepts often to actively suppress interference from similar, lower-priority ones

Emotion, Stress & Memory

Glucocorticoid modulation (Sapolsky)

- Acute stress (exam, challenge questions) → cortisol → amygdala activation → enhanced hippocampal encoding IF stress ends before retrieval
- Chronic stress → sustained cortisol → hippocampal dendritic pruning → impairs spatial/declarative memory
- Key: stress during encoding helps; stress during retrieval hurts

Emotional tagging (McGaugh)

- Amygdala acts as a "memory modulator" — emotionally arousing events get a stronger consolidation signal via norepinephrine
- Attaching meaning/stakes to dry facts (e.g., "this is the mechanism behind your patient dying") boosts retention

Sleep — The Most Underrated Variable

Targeted memory reactivation (TMR)

- Playing subtle cues (odors, sounds) associated with studied material during NREM sleep boosts next-day recall by 20-40%
- Practical version: study in a specific scent environment; recreate it during sleep

Sleep spindles and IQ correlation

- Sleep spindle density (11-16 Hz bursts in NREM2) correlates with fluid intelligence and next-day learning capacity
- Spindle density is trainable: exercise, consistent sleep timing, and reduced blue light increase spindle density

Naps

- 20-min nap (NREM1/2): clears hippocampal buffer, restores encoding capacity to morning levels
- 90-min nap (full cycle): includes REM, consolidates procedural + emotional memory
- Optimal nap window: 1-3pm (aligned with circadian alertness trough)

Working Memory & Cognitive Load

Cognitive load theory (Sweller)

- Working memory: ~4 chunks, ~20s without rehearsal
- Intrinsic load (task complexity) + extraneous load (poor presentation) + germane load (schema formation)
- Reduce extraneous load → more capacity for germane load → faster schema building

Chunking and expertise

- Novices see individual facts; experts see patterns/chunks (chess studies: Chase & Simon)
- Learning strategy: actively look for the category a concept belongs to before memorizing its details

Dual-task interference

- Background music with lyrics, notifications, multitasking all compete for the same phonological loop
- Instrumental/brown noise = acceptable; verbal input = impairs verbal encoding

Metacognition — Knowing What You Don't Know

Dunning-Kruger at the neural level

- Anterior prefrontal cortex (aPFC) tracks confidence calibration
- Poor metacognition = mismatch between felt confidence and actual performance = wasted review time
- Fix: forced confidence ratings before every answer (CHITTA can add this as a pre-answer confidence tap)

Judgments of learning (JOLs)

- Immediate JOLs after studying are inflated (fluency illusion)
- Delayed JOLs (after 24h) are well-calibrated predictors of actual retention
- Implication: don't trust how well you think you know something right after studying it

Illusion of explanatory depth (Rozenblit & Keil)

- People vastly overestimate understanding of systems until asked to explain mechanistically
- The Feynman technique breaks this illusion by forcing mechanistic explanation

The Leverage Stack (Ranked by Effect Size)

1. Retrieval practice — effect size d ≈ 0.50-0.80
2. Spaced repetition — d ≈ 0.46
3. Elaborative interrogation — d ≈ 0.44
4. Interleaving — d ≈ 0.42
5. Sleep (7-9h + consistent timing) — foundational; multiplies all others
6. Prediction error / pre-testing — emerging, d ≈ 0.35-0.55
7. Metacognitive monitoring — removes wasted study time
